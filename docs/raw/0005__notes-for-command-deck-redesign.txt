== Command Interface Deck v2 redesign notes ==
date: 2026-06-03

- Commands need preview lines.
- A command summary page should show ~10 command slots.
- Each command row should show:
  command name, command preview, short description, long description.
- Argument configuration should move to a per-command detail page.
- Arguments should have consistent global definitions.
- Same argument should not mean different types in different commands.
- Applicability/requiredness should use:
  blank = unused
  ○ = optional
  ● = required
- The current matrix was structurally correct but visually too hard to read.
- Curved connector lines are beautiful but not sufficient for rapid comprehension.
